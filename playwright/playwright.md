# Playwright Basics (Automation framework)

Anything you wants to automate on website we need to use *playwright*.

Moden website web application automating agent, developed by microsoft. 

More reliable 

## Installation 

pip install playwright 
playwright install chromium 

## Playwright key functionalities 

1. navigation 

How to move website

3. typing or filling forms - form fill

4. waiting
