# API
To communicate between two softwares we use api. 

## API types 
1. REST API
2. GRPAHQL (Meta using, secured one)
3. SOAP API
4. RPC (Remote Procedural Call)
5. Websocket (We use in realtime)


## Rest API and methods  
1. POST  (posting info into the server)
2. GET (getting the info into the server)
3. PUT (update the info on the server)
4. DELETE  (remove the info on the server)
