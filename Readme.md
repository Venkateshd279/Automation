# Automation 

Automation means doing the same tasks over and over automatically, instead of doing them manually each time. 

## RPA 

**RPA (Robotic Process Automation)** means using software robots or bots to do repeated tasks instead of a person. The bot follows rules and can do the same work faster and with fewer mistakes.

### Real-Life Examples:
- Filling forms
- Sending emails
- Creating reports
- Clicking buttons
- Copying and pasting data
- Downloading files

### Benefits:
- **Saves money**: Bots can do work that people do now, so costs can go down
- **Runs any time**: Bots can work day and night without rest

### RPA Tools:
- **PyAutoGUI**: Works on desktop programs and controls the mouse and keyboard
- **Playwright**: Works in web browsers and can control websites automatically

#### PyAutoGUI

PyAutoGUI is a Python package that can move the mouse, click buttons, and type on the keyboard. It works by controlling the screen, so while the bot is running, you usually cannot use the computer for other tasks.

Common actions:
1. Mouse click (left, right)
2. Keyboard typing

#### Steps to install PyAutoGUI

1. Create a virtual environment:
   python -m venv venv
2. Activate it:
   source venv/bin/activate
3. Install PyAutoGUI:
   pip install pyautogui
